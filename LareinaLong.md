## 本周学习内容

### 1. 区块链基础概念复习
- **区块链的定义**：一种去中心化的分布式账本，由多个节点共同维护，数据不可篡改且公开透明。  
- **工作原理**：区块按时间顺序链接形成链条，每个区块包含交易数据和前一区块的哈希值，依赖共识机制（如 PoW）保障安全性。  
- **不可篡改性**：任何篡改都会破坏哈希链结构，从而被网络拒绝。  
- **链的类型与应用场景**：  
  - 公链：开放、透明、任何人可参与（例：比特币、以太坊）。  
  - 联盟链：由特定组织联盟共同维护，适合企业合作（例：供应链金融）。  
  - 私链：由单一机构管理，效率高、隐私性强（例：企业内部管理系统）。  

### 2. 加密货币与激励机制
- **比特币工作量证明（PoW）**：矿工通过计算哈希值竞争记账权，获胜者打包区块并获得比特币奖励。  
- **以太坊账户模型**：分为外部账户（由私钥控制）和合约账户（由代码控制）。  
- **智能合约**：部署在区块链上的自动化程序，条件满足时自动执行。  
- **代币标准**：  
  - ERC-20：同质化代币标准（例：USDT）。  
  - ERC-721：非同质化代币标准（例：NFT）。  

### 3. Web3 典型应用场景
- **DeFi（去中心化金融）**：如 Uniswap（去中心化交易所）、Aave（借贷协议）。  
- **NFT 与数字藏品**：如 OpenSea（NFT 交易市场）。  
- **DAO（去中心化自治组织）**：通过治理代币进行投票决策，实现社区自治。

# 2. 学习了解区块链基础概念
### 一、什么是区块链 & 核心特性
区块链是一种去中心化的分布式账本，由多个节点共同维护、不依赖中心权威，保证数据不可篡改与公开透明。
每个区块包含交易数据与上一块的哈希，串联形成链条；任意篡改都会破坏后续链的完整性。
### 二、比特币机制与代币激励
节点（矿工）通过验证交易、打包区块获得加密代币奖励，这是驱动区块链网络运行的经济机制。
比特币具有稀缺性、匿名性、透明性等属性，但因其网络设计限制确认速度较慢，市场价格波动较大。
### 三、公链、联盟链与私链对比
- 公链：完全开放、任何人可参与，透明度高但效率相对较低，典型应用：比特币、以太坊。
- 联盟链：由预先筛选的组织组成联盟，共同治理，适合企业合作场景如供应链、金融协作。
- 私链：由单一组织控制访问与管理规则，集中权限，高效率高隐私，但缺乏去中心化特征。
### 四、Web3 与 Web2 / Web 3.0 的差异
- Web2：中心化平台主导，用户非数据拥有者（如微信、抖音）。
- Web 3.0（语义网）：强调机器可读数据与语义结构化，不依赖区块链技术。
- Web3：基于区块链，用户拥有身份和资产，智能合约自动执行规则，支持 DeFi、NFT、DAO 等去中心化应用。
